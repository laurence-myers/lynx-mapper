This library targets Deno:

```
deno 2.0.3 (stable, release, x86_64-pc-windows-msvc)
v8 12.9.202.13-rusty
typescript 5.6.2
```

## Deno usage

- Type check: `deno check .`
- Format (check): `deno fmt --check`
- Format (write): `deno fmt`
- Lint: `deno lint`
- Test: `deno test`

Do all checks:

```powershell
deno task check-all
```
